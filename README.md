# SpeakFlow - 影子跟读学习工具

专为语言学习者设计的 Chrome 扩展，支持多语言、多音色、倍速播放和循环练习。

## 功能特点

- 🎤 **多语言支持**：中文、日语、英语、韩语、法语、德语、西班牙语等多种语言
- 🎵 **多音色选择**：每种语言提供多个不同音色可选
- ⚡ **倍速播放**：支持 0.5x - 2.0x 倍速调节
- 🔁 **循环播放**：支持单次或循环播放，关闭弹窗也能继续播放
- 💾 **设置记忆**：自动保存您的语言、音色、倍速等设置
- 📱 **后台播放**：使用 Service Worker，关闭扩展弹窗也能继续播放

### 方式二：开发者模式安装

1. 下载或克隆本仓库：
   ```bash
   git clone https://github.com/lxy20210816-netizen/speakflow.git
   ```

2. 打开 Chrome 浏览器，访问 `chrome://extensions/`

3. 开启右上角的"开发者模式"

4. 点击"加载已解压的扩展程序"

5. 选择 `chrome-extension` 文件夹

6. 安装完成！点击浏览器工具栏中的扩展图标即可使用

## 使用方法

1. **打开扩展**：点击浏览器工具栏中的 SpeakFlow 图标

2. **输入文字**：在文本框中输入或粘贴要练习的文字

3. **选择语言**：从下拉菜单中选择目标语言

4. **选择音色**（可选）：如果需要特定音色，从音色列表中选择

5. **调节倍速**：拖动滑块设置播放速度（0.5x - 2.0x）

6. **设置循环**：勾选"循环播放"复选框

7. **开始练习**：点击"生成并播放"按钮

8. **停止播放**：点击"停止"按钮

## 技术栈

- **Chrome Extension Manifest V3**
- **Chrome TTS API**：语音合成
- **Service Worker**：后台播放支持
- **Chrome Storage API**：设置持久化

## 注意事项

- 本扩展使用 Chrome 内置的 TTS（Text-to-Speech）引擎
- 语音质量取决于系统安装的 TTS 引擎
- 如需更高质量的语音，建议：
  - Windows：安装高质量语音包
  - macOS：使用系统语音设置优化
  - Linux：安装 `espeak` 或 `festival` 等 TTS 引擎

## 常见问题

### Q: 为什么某些语言没有音色可选？
A: 音色列表取决于系统已安装的 TTS 引擎。可以在系统设置中安装更多语言包。

### Q: 循环播放为什么只播放一次就停止了？
A: 请确保：
   - 勾选了"循环播放"复选框
   - 没有关闭浏览器或禁用扩展
   - 系统 TTS 引擎正常工作

### Q: 可以在后台播放吗？
A: 可以！扩展使用 Service Worker，关闭弹窗后仍能继续播放。
